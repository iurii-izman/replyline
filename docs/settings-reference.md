# Settings Reference

Slim Stable Beta settings:

- `hotkey` (`string`)
- `captureMaxSeconds` (`number`, 5..180, default `45`)
- `llmBaseUrl` (`string`)
- `llmModel` (`string`)
- `deepgramApiKey` (stored as secret)
- `llmApiKey` optional (stored as secret)

No Advanced Mode fields in user-facing settings contract.
