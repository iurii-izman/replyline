# Replyline v0.2.0-beta.1

This is the first public **source/developer beta** of Replyline.

Replyline is a Windows-first desktop tool for difficult live work conversations. Hold
`Ctrl+Alt+Space` during a short fragment, release it, and receive one compact response card:
`gist / say_now / next_move`.

## What Is Ready

- WorkConversation path with bounded `CardSchemaV3` output.
- Interview Mode with Candidate Pack preparation and local report flow.
- Windows tray lifecycle with a normal taskbar and Alt+Tab window.
- Deepgram STT and user-configured OpenAI-compatible LLM routing.
- Key storage through Windows Credential Manager.
- Automated privacy, contract, UI, Rust, dependency, and release-freeze gates.

## How To Try It

This prerelease does not include a public unsigned installer. Build from source:

```powershell
git clone https://github.com/iurii-izman/replyline.git
Set-Location replyline
git checkout v0.2.0-beta.1
pnpm install --frozen-lockfile
pnpm tauri dev
```

Prerequisites:

- Windows 10 or 11
- Node.js 20+
- pnpm 9.15.9
- Rust stable and Tauri Windows build prerequisites
- a Deepgram API key
- an OpenAI-compatible LLM endpoint and model

## Validation

The release commit passed:

- `pnpm verify:full`
- 179 frontend/UI tests
- Rust test, clippy, fmt, and dependency advisory gates
- credential-free web E2E
- CodeQL for Actions, JavaScript/TypeScript, and Rust
- GitHub secret scanning and Dependabot with zero open alerts

## Privacy And Trust

- API keys are stored in Windows Credential Manager.
- Capture audio is processed in memory and is not saved by default.
- Configured STT and LLM providers receive audio/text required for the request.
- Explicit report and evidence commands can create sensitive local artifacts.
- Replyline is visible desktop software and does not provide hidden-mode or control-bypass behavior.

## Known Limitations

- Cross-machine and cross-call-app behavior remains pending broader verification.
- Provider latency and availability vary.
- Product UX is Russian-first.
- There is no signed public Windows installer in this prerelease.
