## [0.1.0-alpha] - 2026-04-10

### Добавлено

- Базовая русскоязычная документация проекта и операционные runbook-материалы.
- Artifacts mirror для артефактов и инженерных следов проверок.
- Интеграционные контуры `bridges/replyline`.
- Операционные скрипты для локального запуска, проверок и поддержки alpha-процессов.
- `docker-compose` окружения для Langfuse и Qdrant.
- Python-демо сценарии для быстрого тестирования AI-потоков.
- Конфигурация `promptfoo` для проверки prompt/eval контрактов.
- GitHub issue/PR templates.
- Governance-документы проекта.

### Инфраструктура

- LiteLLM gateway для унифицированного доступа к моделям.
- Ollama local models для локальных inference-сценариев.
- n8n workflow automation для оркестрации процессов.
- Langfuse observability для трассировки и мониторинга AI-вызовов.
