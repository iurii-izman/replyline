Source-only alpha · Windows-first · Russian-first

Когда ответить трудно, Replyline даёт следующий ход.

Tray-приложение для сложных рабочих разговоров: одна горячая клавиша, короткий фрагмент, одна карточка `gist / say_now / next_move`.

Что это

  • Помощь в моменте, а не пост-разбор звонка.
  • Захват только пока удерживается hotkey.
  • Фокус на коротком ответе, который можно произнести вслух.

Что это не есть

  • Не meeting assistant.
  • Не transcript tool.
  • Не speaking coach.

Trust

Live-фрагменты обрабатываются в RAM и по умолчанию не сохраняются на диск. После отпускания hotkey фрагмент отправляется во внешние STT/LLM провайдеры, которые вы настраиваете сами. Пользователь отвечает за соблюдение правил платформ, политики работодателя и законов о записи.

Поведение между разными машинами и всеми call-приложениями остаётся pending verification.

GitHub (исходники)

Быстрый старт

  1. Клонируйте репозиторий: git clone git@github.com:iurii-izman/replyline.git
  2. Установите зависимости: pnpm install
  3. Запустите dev-режим: pnpm tauri dev
  4. Откройте настройки — введите ключи Deepgram и LLM-шлюза
  5. Удерживайте горячую клавишу во время разговора

Системные требования

  • Windows 10/11 (основная платформа)
  • Node.js 20+, pnpm 9+, Rust toolchain (stable)
  • Ключ Deepgram API (STT)
  • Доступ к OpenAI-совместимому LLM API (OpenRouter, локальный прокси и т.д.)
