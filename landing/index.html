Один bounded flow вместо ещё одного dashboard.

Replyline не записывает весь звонок и не строит историю встреч. Он помогает пройти один конкретный сложный момент.

Это не meeting assistant, не transcript tool и не speaking coach. В текущем beta scope нет скрытого overlay, полной истории разговоров или командных workflows.

01 / CAPTURE

Удерживайте hotkey

Захватывается короткий фрагмент системного аудио, пока клавиша нажата.

02 / ANALYZE

STT и LLM по вашей конфигурации

Deepgram и OpenAI-compatible endpoint настраиваются пользователем.

03 / RESPOND

Одна компактная карточка

gist / say_now / next_move без transcript UI и перегруженного workflow.

Trust boundaries

Без скрытых обещаний.

Поведение продукта описывается через конкретный data flow, а не через расплывчатое «полностью приватно».

  • API keys хранятся в Windows Credential Manager.
  • Capture не сохраняется на диск по умолчанию.
  • Настроенные STT/LLM провайдеры получают необходимые audio/text данные.
  • Evidence и report artifacts создаются только в явных диагностических flows.
  • Это видимое desktop-приложение без скрытого режима или обхода контроля.

Для тестеров и contributors.

Самый полезный вклад сейчас: воспроизводимые Windows-наблюдения, понятные setup notes и небольшие изменения с проверками.

Запустить из source

  1. git clone https://github.com/iurii-izman/replyline.git
  2. git checkout v0.2.0-beta.1
  3. pnpm install --frozen-lockfile
  4. pnpm tauri dev

Нужны Windows 10/11, Node.js 20+, pnpm 9, Rust stable, Deepgram key и OpenAI-compatible LLM endpoint.
